SatTrack: Orbital Mechanics & Real-Time Visualization

  1. Initialize the Backend Environment

    python3 -m venv venv
source venv/bin/activate

    Set up a dedicated Python environment to handle high-frequency orbital calculations and API routing.

  2. Install Core Propagation Engines

    pip install fastapi uvicorn sgp4 numpy

    Beyond standard web tools, we integrate sgp4 for precise position and velocity estimation from TLE (Two-Line Element) sets.

  3. Implement Physics-Based State Estimation

    To solve the 3-second polling 'jitter,' we implement dead reckoning. Instead of jumping to new coordinates, the app predicts the satellite's path using its velocity vector, turning static data into smooth 60fps animations.

  4. Configure Geodetic Math & Map Projections

    Correcting for Mercator distortion, especially for polar-orbiting satellites, requires cosine-corrected heading vectors. This ensures satellites point exactly along their ground track relative to the Earth's curvature.

  5. Launch the Vue 3 Frontend

    cd frontend
npm install
npm run dev

    The UI is optimized for a dark-matter aesthetic using Leaflet.js, providing a high-contrast 'Command Center' feel for global tracking.

  6. Mobile-First PWA Optimization

    Optimized as a Progressive Web App (PWA), SatTrack allows you to track satellites passing overhead directly from a mobile home screen with minimal latency.

Final Capabilities

The system successfully processes real-time TLE data from CelesTrak, calculates orbital paths 60 minutes into the future, and renders them with sub-millisecond precision.
