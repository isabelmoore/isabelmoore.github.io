SpotDL and Mixxx Installation Guide

  1. Install SpotDL

    python3 -m pip install spotdl

    This installs the SpotDL package (Spotify downloader) into your Python environment.

  2. Create a Directory for Downloads

    
mkdir dj_songs
cd dj_songs/

    Create a new directory called dj_songs and navigate into it to organize your downloads.

  3. Install Mixxx

    sudo apt install mixxx

    This command installs Mixxx, a DJ software, and its dependencies.

  4. Attempt to Download a Spotify Playlist

    python3 -m spotdl <Spotify Playlist URL>

    You attempt to download songs from the playlist, but an error occurs because FFmpeg is not installed:

    FFmpegError: FFmpeg is not installed.

  5. Download FFmpeg Using SpotDL

    spotdl --download-ffmpeg

    This downloads and installs FFmpeg, which is required for SpotDL to work.

  6. Retry Downloading the Spotify Playlist

    python3 -m spotdl <Spotify Playlist URL>

    SpotDL processes the playlist URL, finds the songs, and downloads them as audio files.

Final Output

The downloaded songs are saved in the dj_songs directory, ready for use in Mixxx or any other media player.
